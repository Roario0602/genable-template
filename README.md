## üá∫üá∏ Description

**Genable Template** is a minimal Android template written in **Kotlin**,
designed for building simple character-based applications.

By default, the app:
- displays a **white background**
- shows a **centered PNG character image**
- plays **looping background music**

This project is a **template**, not a finished product.

‚ùó It is **NOT an original app**,  
‚ùó **NOT a clone** and **NOT a continuation** of any existing project,  
‚ùó **NOT affiliated** with any authors or characters.

All images and audio files included are **placeholders**
and are intended to be replaced.

---

## üéØ Application Behavior

- single main screen
- white background
- centered PNG character
- background music starts on launch
- music continues when the app is minimized
- music stops only when the app is fully closed
- no network usage
- no dangerous permissions by default

## ‚úèÔ∏è Customization

To create your own app, simply replace the following files:

- `app/src/main/res/drawable/character_example.png`
- `app/src/main/res/raw/background_music_example.mp3`

File names must remain unchanged.
